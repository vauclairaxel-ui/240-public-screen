intro

Bienvenue sur ce mini diaporama

Cette page fonctionne entièrement en local. Elle peut être copiée sur une clef USB, ouverte hors connexion ou intégrée dans un dossier partagé. Les slides défilent automatiquement pour offrir une boucle continue d’informations.

concept

Structure simple, contenu libre

Chaque slide est un simple bloc HTML (div) avec du texte, des images ou tout autre contenu. Le JavaScript embarqué s’occupe de les afficher l’un après l’autre. Tu peux dupliquer une slide, adapter les styles et jouer avec la mise en page pour raconter une histoire, présenter un produit ou partager un plan de cours.

bonus

Animation fluide côté desktop

Les transitions sont gérées en CSS pour conserver de bonnes performances, et le script fait défiler automatiquement les slides toutes les cinq secondes. Idéal pour un poste fixe ou un poste kiosque qui diffuse une boucle d’informations sans dépendre de frameworks extérieurs.
